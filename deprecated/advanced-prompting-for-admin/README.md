## 非推奨の理由

v0.1.0 のロードマップでは、大学職員向けに「応用プロンプト集」を独立スキルとして提供する計画でした。しかし v0.3.0 で [staff-ai-literacy-primer](../../skills/staff-ai-literacy-primer/) をリリースした段階で、新任職員が最初に必要とするプロンプトの型（指示の具体化、役割付与、出力形式の指定、評価と反復）は同スキルの本体でほぼ包含されることが判明しました。

独立した「応用プロンプト集」は、粒度として中途半端になります。基礎リテラシーからは切り出し過ぎており、一方で業務領域別のプロンプト例（シラバス、広報、IR など）は `domain-skills/` 配下の各スキルに分散して収録する方が文脈保持の観点から自然です。このため、単独スキルとしての続行は断念し、機能は既存スキルに吸収する判断としました。

## 代替スキル

- **[skills/staff-ai-literacy-primer](../../skills/staff-ai-literacy-primer/)** — 新任職員向け AI リテラシー入門。プロンプト設計の基本型と SD 研修のコア教材を兼ねます。応用的な使い方は、本スキルで想定していた内容を吸収しています。
