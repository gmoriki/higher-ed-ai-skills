# 例 1: 広報原稿 (AI 臭多め) の評価

## 入力

```
/ai-tone-check /tmp/press-release-draft.md public
```

### ドラフト本文 (架空、AI 生成の匂いを意図的に残した版)

```markdown
# ◯◯大学、AI を活用した入試説明会を開催

## 概要

近年、AI 技術の急速な発展により、教育現場においても生成 AI の活用が
重要なテーマとなっています。このような状況の中、◯◯大学では
AI を活用した画期的な入試説明会を開催することとなりました。

本説明会は、受験生の皆様にとって大変有意義な機会となることが
期待されており、参加者には個別最適化された情報を提供することが
できます。

## 開催の意義

一般的に、入試説明会は画一的な情報提供に留まることが課題とされて
います。本学では、AI を活用することで、受験生一人ひとりに寄り
添った案内を実現し、素晴らしい体験を提供いたします。

また、デジタル技術の活用は時代の要請であり、本学の教育改革の一環
として位置付けられています。

## 参加方法

以下の方法でお申し込みいただけます。

- 大学公式 Web サイトからの申込
- 電話での申込
- 郵送での申込

皆様のご参加を心よりお待ちしております。
```

## skill の処理

1. 入力ファイルを Read
2. 機密度チェック: 学籍番号 / 個人メール / 氏名なし → OK
3. ロール = `public` (学外読者、受験生の保護者 / 地域住民 / 企業担当者想定)
4. `Task` で general-purpose subagent を 1 体 dispatch、SKILL.md 手順 3 の template を埋めて渡す
5. subagent が 3 軸採点 + 指摘 Top 5 + 総評を返す
6. 親 skill が整形してユーザーに提示

## 期待される出力

```markdown
## AI Tone Check 結果

**ドラフト:** /tmp/press-release-draft.md
**ロール:** public
**総合スコア:** 17/30 (A: 7/10, B: 3/10, C: 7/10)

### 指摘 Top 5

1. L5: "AI 技術の急速な発展により、教育現場においても生成 AI の活用が重要なテーマとなっています" → 典型的な AI 臭の時事背景導入。削除するか、具体的な本学の文脈 (「本学では 2024 年度から生成 AI 活用方針を策定してきた」等) に置換
2. L8: "画期的な入試説明会" → 空虚強調 (「画期的」)。何が具体的に変わるのか (例: 個別質問への AI 応答、志望学部ごとのオーダーメイド資料) で置換
3. L11: "大変有意義な機会となることが期待されており" → 冗長敬語。「参加者にとって有益な時間になる」「個別最適化された情報を得られる」等の具体表現に
4. L18: "一般的に、入試説明会は画一的な情報提供に留まることが課題とされています" → 主語の一般化 + 抽象論。「従来の説明会では ◯◯ の声を多くいただいていた」等、本学の実績や調査結果に基づく記述に
5. L22: "素晴らしい体験を提供いたします" → 空虚強調 (「素晴らしい」)。何が素晴らしいのか具体的に (「参加者は事前に志望学部を登録すれば、当日の案内が自動で絞り込まれます」等)

### 総評

学外読者として読むと、本学が実際に何を新しくするのかが最後まで分からない。
「AI を活用した」が形容詞として繰り返されるだけで、受験生が当日どう違う体験を
するのか、例えば「個別質問に AI が即答する」「志望学部別の資料が自動生成される」
等の具体が欠けている。空虚強調と時事背景の導入で全体の 1/3 を占めており、残り
2/3 に移せる具体情報の余地がある。受験生の保護者が「参加すべきか」を判断できる
材料として機能していないため、総合スコアは中位だが修正優先度は高い。

### 次のアクション候補

1. 「画期的」「素晴らしい」「大変有意義」の空虚強調 3 箇所を削除し、実際の新規性 (AI が何をするか) を具体文に置換する
2. 冒頭の時事背景導入 (L4-6) を削除し、本学の具体施策から書き出す
3. 参加方法の箇条書き (L30-32) の後に「申込後の流れ」を 1-2 文追加 (読者負荷の C 軸改善)

---
※ 本 skill は単発実行。修正後の再評価が必要なら再度 `/ai-tone-check` を呼んでください。
```

## なぜこの応答が適切か

- 指摘は全て行番号 + 引用 + 改善案の 3 要素揃い (曖昧な「気をつける」が無い)
- スコアだけでなく「致命的な指摘 = 何も新しくないように読める」を総評で明示
- 広報原稿として修正する場合の優先順位が明確 (空虚強調の削除から始める)
- 「学外読者として読むと」と主語が明示され、ロール評価として機能
- B 軸 (AI 臭度) が 3/10 と低く、該当パターンが多いことが定量的に示される
- 「AI 臭ゼロを追うと大学らしい硬さまで削れる」前提で、B 軸 7-8/10 を目安に改善を案内する余地を残している
